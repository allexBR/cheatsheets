##-------------##
##  UFW Rules  ##
##-------------##

1. Network Redirection (NAT) Configuration:

nano /etc/ufw/before.rules

*nat
:PREROUTING ACCEPT [0:0]

# Redirects packets arriving at eth0 (port 53) to the local IP address 127.0.0.1
-A PREROUTING -i eth0 -p udp --dport 53 -j DNAT --to-destination 127.0.0.1:53
-A PREROUTING -i eth0 -p tcp --dport 53 -j DNAT --to-destination 127.0.0.1:53

COMMIT



2. Allow traffic through the firewall (allow redirect traffic to reach Unbound):

ufw allow in on eth0 to 127.0.0.1 port 53 proto udp

ufw allow in on eth0 to 127.0.0.1 port 53 proto tcp



3. Important Note!

# For redirection to work, make sure IP Forwarding is enabled on your Linux system.
# You can check this with:
sysctl net.ipv4.ip_forward

# If the value is 0, enable it by editing the /etc/sysctl.conf file and uncommenting the line:
net.ipv4.ip_forward=1

# Then apply it with:
sudo sysctl -p
